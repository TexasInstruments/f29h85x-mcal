Documentation
==============
1. Release notes is available at {install_directory} - docs/User_Guide/src/mcal_release_notes.html
2. MCAL User Guide is located at - {install_directory}/docs/User_Guide/index.html. Online version of User Guide is also available at - https://software-dl.ti.com/C2000/docs/mcal/F29H85x/latest/index.html.
3. Test Reports for each module are available at - {install_directory}/docs/Test_Report

Steps to build
==============
1. Configure CMake with your choice of Generator (Refer Chapter 3.11 in User Guide for more details)
2. Build using CMake (Refer Chapter 3.11 in User Guide for more details)

Notes
==============
1. EB Tresos Plugin (located at - {install_directory}/plugins/) Com_TI_F29H85x, Dem_TI_F29H85x, EcuM_TI_F29H85x, Os_TI_F29H85x are stubs used for MCAL BareMetal examples and should not be used in typical AUTOSAR Application.
