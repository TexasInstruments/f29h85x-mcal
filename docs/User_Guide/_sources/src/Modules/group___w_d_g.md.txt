### enum Wdg_StatusType
| Enumerator | Value | Description |
| ---------- | ----- | ----------- |
| WDG_UNINIT | 0U| Watchdog driver is uninitialized. State on power on before invoking Wdg_Init()   |
| WDG_IDLE | 1U| Watchdog driver is in idle state. Default state when WDG timer regs are not being programmed.   |
| WDG_BUSY | 2U| Watchdog driver is reprogramming the WD to trigger the watchdog.   |




**Brief**: 
This is a Watchdog status type value. 



### enum Wdg_Reaction
| Enumerator | Value | Description |
| ---------- | ----- | ----------- |
| WDG_GENERATE_RESET | 0U| Watchdog can generate a reset signal.   |
| WDG_GENERATE_INTERRUPT | 1U| Watchdog can generate an interrupt signal; reset signal is disabled.   |




**Brief**: 
Watchdog module Reaction if watchdog counter reaches the maximum value.

Wdg_PreDivider determines the watchdog clock pre-divider, which is the first of the two dividers between INTOSC1 and the watchdog counter clock (WDCLK)
