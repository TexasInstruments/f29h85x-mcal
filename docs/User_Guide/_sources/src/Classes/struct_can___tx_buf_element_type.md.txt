# Can_TxBufElementType



Structure for CAN Tx Buffer element. 


`#include <Can_Priv.h>`

## Structure Members

|  Member Type   | Member name    | Brief |
| -------------- | -------------- | ----- |
|  uint32 | id |  Identifier.  |
|  uint8 | xtd |  Extended Identifier 0 = 11-bit standard identifier 1 = 29-bit extended identifier.  |
|  uint32 | dataLength |  Data Length Code 0-8 = CAN CAN FD transmit frame has 0-8 data bytes 9-15 = CAN FD transmit frame has 12/16/20/24/32/48/64 data bytes.  |
|  boolean | brs |  Bit Rate Switching 0 = CAN FD frames transmitted without bit rate switching 1 = CAN FD frames transmitted with bit rate switching.  |
|  boolean | fdf |  FD Format 0 = Frame transmitted in Classic CAN format 1 = Frame transmitted in CAN FD format.  |
|  Can_MemType | memType |  Can Message Ram Type.  |
|  uint8[((uint8) 64U)] | data |  Data bytes. Only first dlc number of bytes are valid.  |
